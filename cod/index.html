Estruturação basica do HTML com suas tags

(Inicio)

Um documento HTML começa com uma declaração do tipo de documento, que informa ao navegador que o documento é um arquivo HTML

Todo o conteúdo da página é encapsulado em um elemento "html", este elemento serve como o contêiner raiz de toda a página

A seção "head" da página contém informações sobre a página que não são visíveis para os usuários, mas são importantes para os navegadores e mecanismos de busca. Alguns elementos comuns encontrados no "head" incluem:

  • "meta": Define metadados, como a codificação de caracteres e a descrição da página.
  • "title": Define o título da página exibido na barra de título do navegador.
  • "link": Usado para vincular folhas de estilo externas (CSS).
  • "script": Pode incluir scripts JavaScript e outros recursos.

O elemento "body" contém todo o conteúdo visível da página, como texto, imagens, links, formulários, etc. É aqui que a maior parte do conteúdo da página é inserida.

Atributos de tags

(Inicio)

Os atributos de tags são componentes vitais do HTML que permitem que os desenvolvedores forneçam informações adicionais sobre os elementos HTML e controlem seu comportamento e estilo. Eles são especificados dentro das tags e são compostos por um nome e um valor, separados por um sinal de igual (=).

Alguns exemplos comuns de atributos incluem:

  • "href": É usado com a tag "a" para definir o destino de um link.
  • "src":Se utiliza juntamente com a tag "img" para especificar o caminho da imagem

    • É importante notar que diferentes elementos HTML têm atributos específicos que se aplicam a eles, e esses atributos desempenham um papel fundamental na definição do comportamento e da apresentação desses elementos
